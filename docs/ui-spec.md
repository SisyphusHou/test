# 五子棋 UI 设计规范

版本: 1.0 | 日期: 2026-06-16

## 1. 设计理念

深色主题（深蓝灰基底），木纹棋盘为视觉焦点，克制的半透明分层，即时反馈。

## 2. 色彩系统

- 页面底: #1a1a2e | 面板底: #16213e | 卡片底: #1e1e3a
- 文字主: #e0e0e0 | 文字次: #a0a0b0 | 文字 muted: #888
- 边框: rgba(255,255,255,0.08~0.30) 分层
- 语义: 胜线描金 rgba(255,215,0,0.9) | 最后落子红 #ff4d4d
- 棋盘木纹: #deb887 → #d2a66e → #c49a5a
- 黑子: radial(#555→#1a1a1a→#000) | 白子: radial(#fff→#e8e8e8→#bbb)

## 3. 字体系统

系统原生栈: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif。
字号: 13px (分段/按钮) | 15px (回合提示) | 22px (获胜标题)。字重: 500 / 600。

## 4. 组件规范

### 分段控件
圆角 10px，内边距 3px，选项 8×18px，激活态 rgba(255,255,255,0.10) 底色。

### 标准按钮
7×14px 内边距，圆角 8px，hover 底色加深，active scale(0.97)，disabled opacity 0.4。

### 主要按钮
10×28px 内边距，圆角 10px，14px 字号，600 字重。

### 棋盘 Canvas
15×15 路，32px 边距，36px 间距，棋子半径 15.84px。木纹底→网格线→星位→棋子→胜利线→红点→悬停预览。

## 5. 响应式

唯一断点 640px：controls-row 纵向，info-panel 纵向左对齐，分段按钮缩小。

## 6. 圆角系统

8px(按钮/棋盘) → 10px(分段控件) → 12px(面板) → 16px(弹窗卡片)，圆形 50%。
