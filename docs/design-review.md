# 五子棋 UI 设计评审报告

评审日期: 2026-06-16 | 评级: B+

## 发现项

### P1 — 应尽快修复
- **[P1-1]** 焦点指示器全局缺失: `outline: none` 全局生效，键盘用户无法判断焦点 — 改为 `:focus-visible` 可见指示
- **[P1-2]** 获胜弹窗缺少语义通知 — 添加 `role="alert"` 和 `aria-live="assertive"`
- **[P1-3]** Canvas 无替代文本 — 添加 `role="img"` + `aria-label`
- **[P1-4]** 小屏画布溢出 — 568px 固定宽度在 320-375px 手机上超出视口

### P2 — 建议优化
- 缺少 CSS 自定义属性（设计变量），色值硬编码不利主题切换
- 未激活分段按钮对比度 #888 在 #16213e 上约 3.9:1，低于 WCAG AA 4.5:1
- 缺少平板横屏断点（仅 640px 一个）
- 禁用按钮仍会缩放（`.btn:active` 独立于 disabled）
- Escape 键无法关闭胜利弹窗
