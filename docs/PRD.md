# 五子棋 — 产品需求文档 (PRD)

版本: 1.0 | 日期: 2026-06-16 | 状态: 评审通过

## 1. 产品概述与目标

五子棋（Gomoku）是一款基于浏览器的单页面五子棋对弈应用，零门槛开箱即玩。支持人人对战和人机对战，AI 基于 Minimax + Alpha-Beta 剪枝，提供简单/一般/困难三档难度。

**目标用户**: 休闲玩家、五子棋学习者、同屏双人对弈者。

## 2. 用户故事

| 编号 | 用户故事 | 验收标准 |
|------|----------|----------|
| US-01 | 打开页面即见棋盘可对弈 | 1 秒内加载，默认人人模式 |
| US-02 | 与不同水平 AI 对弈提升棋力 | 三档难度，切换自动重置 |
| US-03 | 落子前预览避免误触 | 鼠标悬停半透明预览 |
| US-04 | 追踪上一手位置 | 红色圆点标记最后落子 |
| US-05 | 支持悔棋与快捷键 | 按钮 + Ctrl+Z，人机撤销双方各一手 |
| US-06 | 获胜时明确提示 | 金色高亮连线 + 弹窗 |
| US-07 | 跨局累计比分 | 会话内累加，切换模式不重置 |
| US-08 | 手机触屏适配 | 640px 断点纵向布局 |

## 3. 功能需求 (MoSCoW)

### Must Have — v1.0 已交付 (14 项)
15×15 Canvas 棋盘、人人/人机双模式、三档 AI 难度、落子交互、胜负判定、胜利弹窗、悔棋、重新开始、计分板、回合指示、最后一手标记、触屏支持、响应式布局。

### Should Have — v1.1 规划
棋局回放、音效反馈、设置持久化(localStorage)、平局优化、棋盘尺寸选择(13/15/19)、禁手规则、对局计时。

### Could Have — v2.0 规划
在线对战(WebSocket)、排行榜(Elo)、用户系统、观战模式、PWA、AI 难度连续调节、新手教程、主题切换、SGF 棋谱导出。

## 4. 非功能需求

- **性能**: 首屏 ≤1s，AI 简单 ≤50ms / 一般 ≤500ms / 困难 ≤3s
- **兼容**: Chrome/Firefox/Safari/Edge 90+，Windows/macOS/iOS/Android
- **安全**: v1.0-v1.1 无服务端依赖，零数据泄露风险
- **可维护**: IIFE 作用域隔离，零构建工具依赖

## 5. 版本路线图

- **v1.0** (2026-06 已上线): 核心对弈闭环
- **v1.1** (Q3 规划): 体验深化 — 回放/音效/持久化/禁手
- **v2.0** (Q4 规划): 社交联机 — 在线对战/排行榜/PWA
