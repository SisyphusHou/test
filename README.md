# 五子棋

基于 Canvas 的单页面五子棋游戏，支持人人对战和人机对战，带 AI 难度选择。

## 功能

- **人人对战** — 同屏双人对弈
- **人机对战** — Minimax + Alpha-Beta 剪枝 AI，三档难度可选
- **悔棋** — 支持 Ctrl+Z 快捷键，人机模式下撤销双方各一手
- **计分** — 跨局累计黑白胜负场
- **触屏支持** — 手机和平板可玩

## 难度

| 难度 | 搜索深度 | 特点 |
|------|---------|------|
| 简单 | 2 层 | 随机性走法，适合新手 |
| 一般 | 4 层 | 攻守平衡 |
| 困难 | 6 层 | 深层次搜索，走法缜密 |

## 使用

直接浏览器打开 `gomoku.html` 即可，无依赖。

```
git clone https://github.com/SisyphusHou/test.git
cd test
# 浏览器打开 gomoku.html
```

## 技术

- HTML5 Canvas 棋盘渲染
- Minimax 算法 + Alpha-Beta 剪枝
- 五连窗口评估函数，带防守加权
- 候选走法邻域剪枝与排序优化
